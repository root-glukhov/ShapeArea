# ShapeArea

### Реализация
Для поддержки гибкости и расширяемости кода все фигуры наследуются от базового (абстрактного) класса `Shape` который определяет абстрактное свойство `Area`, которое должно быть реализовано в каждом конкретном классе фигуры. Сам же базовый класс наследуется от интерфейса `IShape`, который определяет общее свойство `Area` для всех фигур. Это обеспечивает абстракцию, которая позволяет работать с любыми фигурами, не заботясь о деталях их реализации.

### Тестирование
Все функции в этой библиотеке тщательно протестированы с использованием юнит-тестов (xUnit).

#### `CircleTests` проверяют:
- что площадь окружности вычисляется правильно.
- что при попытке создать окружность с отрицательным радиусом выбрасывается исключение ArgumentException.

#### `TriangleTests` проверяют:
- что площадь треугольника вычисляется правильно.
- что свойство IsRightAngled верно определяет является ли треугольник прямоугольным.
- что при попытке создать треугольник с отрицательными сторонами выбрасывается исключение ArgumentException.
- что при попытке создать треугольник с недопустимыми сторонами (сумма двух сторон меньше третьей) выбрасывается исключение ArgumentException.
